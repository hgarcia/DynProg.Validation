h3. Methods

__All the methods generate an InvalidSizeException with a message detailing the problem according to the check performed.__

CheckDoubleCondition IsBetween(double lowerBound, double upperBound)

CheckDoubleCondition IsEqualTo(double compareValue)

CheckDoubleCondition IsLessOrEqualTo(double compareValue)

CheckDoubleCondition IsLessThan(double compareValue)

CheckDoubleCondition IsMoreOrEqualTo(double compareValue)

CheckDoubleCondition IsMoreThan(double compareValue)

h3. Generic methods

__All generic methods will generate an Exception of type E. E must inherit from Exception. This methods always take an array of objects with the parameters for E.__

CheckDoubleCondition IsBetween&lt;E&gt;(double lowerBound, double upperBound, object[] exceptionArguments) where E : Exception

CheckDoubleCondition IsEqualTo&lt;E&gt;(double compareValue, object[] exceptionArguments) where E : Exception

CheckDoubleCondition IsLessOrEqualTo&lt;E&gt;(double compareValue, object[] exceptionArguments) where E : Exception

CheckDoubleCondition IsLessThan&lt;E&gt;(double compareValue, object[] exceptionArguments) where E : Exception

CheckDoubleCondition IsMoreOrEqualTo&lt;E&gt;(double compareValue, object[] exceptionArguments) where E : Exception

CheckDoubleCondition IsMoreThan&lt;E&gt;(double compareValue, object[] exceptionArguments) where E : Exception

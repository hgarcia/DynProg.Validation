h3. Methods

__All the methods generate an InvalidSizeException with a message detailing the problem according to the check performed.__

CheckDecimalCondition IsBetween(decimal lowerBound, decimal upperBound)

CheckDecimalCondition IsEqualTo(decimal compareValue)

CheckDecimalCondition IsLessOrEqualTo(decimal compareValue)

CheckDecimalCondition IsLessThan(decimal compareValue)

CheckDecimalCondition IsMoreOrEqualTo(decimal compareValue)

CheckDecimalCondition IsMoreThan(decimal compareValue)

h3. Generic methods

__All generic methods will generate an Exception of type E. E must inherit from Exception. This methods always take an array of objects with the parameters for E.__

CheckDecimalCondition IsBetween&lt;E&gt;(decimal lowerBound, decimal upperBound, object[] exceptionArguments) where E : Exception

CheckDecimalCondition IsEqualTo&lt;E&gt;(decimal compareValue, object[] exceptionArguments) where E : Exception

CheckDecimalCondition IsLessOrEqualTo&lt;E&gt;(decimal compareValue, object[] exceptionArguments) where E : Exception

CheckDecimalCondition IsLessThan&lt;E&gt;(decimal compareValue, object[] exceptionArguments) where E : Exception

CheckDecimalCondition IsMoreOrEqualTo&lt;E&gt;(decimal compareValue, object[] exceptionArguments) where E : Exception

CheckDecimalCondition IsMoreThan&lt;E&gt;(decimal compareValue, object[] exceptionArguments) where E : Exception
